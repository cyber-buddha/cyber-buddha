TryHackMe: Ice

  • My first write up for a CTF. Thanks for checking it out! Here is the link if you would like to try out the machine yourself: https://tryhackme.com/room/ice

  • Ice is a windows machine that has a very poorly secured media server.

Step 0: PREP

  • First thing I do is make a directory for this CTF called iCE. I have a dirctory for all tryhackme material, so I create a directory in their.

Step 1: Reconnaissance

  • This is my go to nmap scan for most CTFs: nmap -A -T4 10.10.54.134 > nmap_scan.txt
  • the greater sign saves the output of the command into a file of your choice. Since I am in a directory meant for the ICE machine I just name the file 'nmap_scan.txt'. Saving the results into a text file allows you to be able have it readily availble in the future, without having to rerun the scan.
  • based on this scan we see a number of open ports, we know that the machine is running windows 7 professional 6.1, and we know that the hostname of the machine is: DARK-PC
  • As you can see the firewall has been disabled on this machine. The ports that you can pay more attention to is port 3389 which is assigned MSRDP (Microsoft Remote Desktop) and port 8000 which is running 'Icecast streaming media server'.
